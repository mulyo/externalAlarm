Install net-snmp Karena EA menggunakan dhcp, cari nomor ip nya di dhcp server Melihat semua oid : snmpwalk -v 2c -c public Setting snmp trap manager : snmpset -v 2c -c private .1.3.6.1.2.1.47.1.1.1.1.19.1 s “192.168.0.107” Dimana 192.168.0.107 adalah ip dari snmp manager Setting snmp trap batas suhu : snmpset -v 2c -c private .1.3.6.1.2.1.47.1.1.1.1.20.1 s “3000” Dimana batas suhu agar trap adalah 30.00 derajat celcius Cara lain untuk test : menggunakan MIB Browser dan PowerSNMP Free Manager yang bisa didownload dari internet
