External Alarm versi 1.03

Untuk melakukan test

1. Install net-snmp

2. ip address dari EA adalah 192.168.1.251

3. Melihat semua oid :

snmpwalk -v 2c -c public 192.168.1.251

4. Setting snmp trap manager :

snmpset -v 2c -c private 192.168.1.251 .1.3.6.1.2.1.61668.1.1.1.1.19.1 s “192.168.1.107”

Dimana 192.168.1.107 adalah ip dari snmp manager

5. Setting snmp trap batas suhu :

snmpset -v 2c -c private 192.168.1.251 .1.3.6.1.2.1.61668.1.1.1.1.20.1 s “3000”

Dimana batas suhu agar trap adalah 30.00 derajat celcius

Cara lain untuk test : menggunakan MIB Browser dan PowerSNMP Free Manager yang bisa didownload dari internet
